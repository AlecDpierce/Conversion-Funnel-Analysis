## Raw Data

This folder is intentionally empty. The actual dataset file size is (5.67GB on disk).

Download `2019-Oct.csv` from Kaggle and place it in this folder before running the notebook:

https://www.kaggle.com/datasets/mkechinov/ecommerce-behavior-data-from-multi-category-store?select=2019-Oct.csv

Once downloaded, the notebook expects the file at:
```
data/raw/2019-Oct.csv
```
